# Pizza Management Backend

A FastAPI-based backend service for managing pizzas, orders, and pricing using RESTful APIs.  
This project is designed to practice backend development concepts, clean project structure, and modern Python dependency management using Poetry.

---

## Overview

The Pizza Management Backend provides a simple and extensible API layer for handling pizza-related operations. It demonstrates common backend patterns such as CRUD operations, request validation, and service execution using FastAPI.

---

## Features

- RESTful CRUD APIs for pizza management
- Order processing with quantity and pricing calculation
- Unique identifiers using UUID
- Lightweight local persistence using SqliteDict
- Automatic interactive API documentation via FastAPI
- Isolated and reproducible dependency management with Poetry

---

## Technology Stack

- Backend Framework: FastAPI  
- Programming Language: Python  
- ASGI Server: Uvicorn  
- Data Storage: SqliteDict  
- Dependency Management: Poetry  
- Testing Framework: Pytest (optional)
